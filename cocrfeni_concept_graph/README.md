Concept Graph with Parallel Processing 


[![meaningtowords](https://img.shields.io/badge/cocrfeniconceptgraph-v1.0-blue)](https://cocrfeni-alloy-concept-graph1.streamlit.app/) ( Keyterms limited to general understanding of MPEA, The script v1 extracts concepts  in parallel from abstracts and so it is faster and runs  without OOM issue for N_doc = 1500 in systems with memory limit of  1 GB, exhibits OOM issues for N_doc >> 1500, Knowledge domain : Generic keyterms on laser and hea interaction, focuse on Alloy Names too, V3 with these features - (Ontology-aware concept resolution (canonicalization via synonym dictionaries), Embedding-based semantic equivalence detection (beyond simple regex), Hierarchical concept taxonomy (hypernym/hyponym relationships), Cross-reference reasoning (linking process parameters to microstructure responses), Entity disambiguation (distinguishing "phase" in thermodynamics vs. metallurgy), Relationship extraction (identifying cause-effect between laser parameters and microstructure) )
